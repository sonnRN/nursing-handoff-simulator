# Disclaimer

## Research Prototype Only

This repository is an AI-assisted nursing handoff demo for research, education, and engineering review.

It is:

- a demo
- a prototype
- a software research artifact

It is not:

- a clinical device
- a diagnosis engine
- a treatment recommendation system
- a production hospital system
- a substitute for clinician judgment

## Not For Clinical Use

Do not use this repository or its outputs to make patient care decisions.

Do not use it in live clinical workflows.

Do not present generated summaries as complete, verified, or medically authoritative.

## Data Scope

This repository is intended to use:

- synthetic local demo data
- public synthetic FHIR sandbox data

If any data source appears ambiguous, realistic, or unsafe for public release, treat it as unsafe and do not commit it.

## Reviewer Note

The purpose of this project is to evaluate whether explainable nursing handoff logic can be built on top of normalized data structures, not to claim readiness for deployment in patient care.
